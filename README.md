# LaDA-Band(3.0B)(Ours)

Anonymous audio demo repository for **LaDA-Band(3.0B)(Ours)**.

This repository hosts selected full-song demos for qualitative comparison against strong baselines.  
All demo links below point to public SoundCloud tracks for more reliable playback and smoother in-browser listening.

Playlists:
- IHP Zeroshot: [anonymous-zeroshot-IHP](https://soundcloud.com/ko9isjyplxrb/sets/anonymous-zeroshot-ihp)
- Suno70K Zeroshot: [anonymous-zeroshot-suno70k](https://soundcloud.com/ko9isjyplxrb/sets/anonymous-zeroshot-suno70k)
- Ablation: [anonymous-ablation](https://soundcloud.com/ko9isjyplxrb/sets/anonymous-ablation)

For double-blind review, this repository currently contains demo audio only.  
Code, checkpoints, and additional materials will be released after the review process.
